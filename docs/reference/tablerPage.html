Create a Tabler Dashboard Page Main function to create a complete dashboard page with Tabler theme

Arguments

titleApp title
navbarDashboard navbar/menu. Can be: - tablerNavbar() for top navigation bar (most layouts) - tablerSidebar() for vertical sidebar (vertical layouts) - horizontalMenu() for horizontal menu (horizontal layout only)
bodyDashboard body content
footerDashboard footer (optional)
layoutLayout type: "boxed", "combo", "condensed", "fluid", "fluid-vertical", "horizontal", "navbar-dark", "navbar-overlap", "navbar-sticky", "rtl", "vertical", "vertical-right", "vertical-transparent"
