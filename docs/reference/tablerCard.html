tablerCard {tabler}R Documentation

Create a Tabler Card

Description

Build a modern card component with optional header, body, and footer

Usage

tablerCard(..., title = NULL, footer = NULL, status = NULL, class = NULL)

Arguments

... Card body content
title Card title (optional)
footer Card footer content (optional)
status Card color status: "primary", "secondary", "success", "warning", "danger", etc.
class Additional CSS classes

Examples

if (interactive()) {
  library(shiny)
  library(tabler)

  ui <- tablerPage(
    title = "Card Demo",
    body = tablerBody(
      tablerCard(
        title = "Sample Card",
        "This is card content",
        footer = "Card footer",
        status = "primary"
      )
    )
  )

  server <- function(input, output, session) {}
  shinyApp(ui, server)
}
