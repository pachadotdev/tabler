Create a Tabler Card Build a modern card component with optional header, body, and footer

Arguments

...Card body content
titleCard title (optional)
footerCard footer content (optional)
statusCard color status: "primary", "secondary", "success", "warning", "danger", etc.
classAdditional CSS classes
